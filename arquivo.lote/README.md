Projeto Java Maven-based prova prática TOTVS
==============

### Instalação

1. Projeto desenvolvido com Java 1.7
2. Rode o maven build com clean install -U -X

### Entrada de dados

1. Adicione os arquivos com a extensão .dat dentro da pasta src/main/resources/dados/in

### Saída de dados

1. O resultado será sumarizado no arquivo sumarizacao.dat.proc dentro da pasta src/main/resources/dados/out

### Testando a aplicação

1. Rode a aplicação executando o método main dentro da classe arquivo.lote.main.Main

### Testes Unitários

1. Foi criado casos de testes na classe br.com.arquivo.lote.TestMotorArquivoLote, utilizando a biblioteca do JUnit para assegurar o funcionamento das regras de tratamentos das informações
